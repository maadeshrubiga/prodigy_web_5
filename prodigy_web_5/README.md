A beautiful, modern weather application that provides real-time weather information with a stunning cosmic design. Built with vanilla HTML, CSS, and JavaScript using the free wttr.in weather API.



## Features

- **City Search** - Search for weather by city name
- **Geolocation** - Use your current location for weather data
- **Temperature Toggle** - Switch between Celsius (¬∞C) and Fahrenheit (¬∞F)
- **Current Conditions** - Display temperature, feels-like, weather description
- **Detailed Metrics** - Humidity, wind speed & direction, visibility, pressure, cloud cover
- **Responsive Design** - Works on desktop and mobile devices
- **Beautiful Animations** - Smooth transitions and floating weather icons

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, animations, glassmorphism effects
- **JavaScript (ES6+)** - Async/await, DOM manipulation, Fetch API
- **wttr.in API** - Free weather data (no API key required)

### Prerequisites
